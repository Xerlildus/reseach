- [ ] **Scale:** ใช้ Redis Cluster สำหรับ Chat และ Job Queue
- [ ] **Security:** ตั้งค่า CORS Whitelist และใช้ Credential Manager ใน n8n
- [ ] **Resource:** กำหนด CPU/RAM Limit สำหรับ Puppeteer และ Video Rendering
- [ ] **Language:** ติดตั้ง Sarabun Font ใน Docker Container สำหรับ PDF/Video
